# Changelog

Все заметные изменения в проекте будут документироваться в этом файле.

Формат основан на [Keep a Changelog](https://keepachangelog.com/ru/1.0.0/),
и этот проект следует [Semantic Versioning](https://semver.org/lang/ru/).

## [0.1.0] - 2026-02-03

### Добавлено
- Начальная версия транскрибатора
- Поддержка транскрибации через faster-whisper
- Определение спикеров через pyannote.audio
- Предобработка аудио/видео файлов
- Экспорт в форматы: текст, JSON, SRT, VTT
- CLI интерфейс с богатыми параметрами
- Работа на CPU без GPU
- Оптимизация для русского языка
- Подробная документация и примеры
- Обработка видео файлов
- Автоматическая конвертация форматов

### Технические детали
- faster-whisper 1.2.1
- pyannote.audio 4.0.3
- PyTorch 2.10
- Python >= 3.10

### Известные ограничения
- Работа на CPU медленнее GPU в 10-20 раз
- Определение спикеров может быть неточным при >5 спикерах
- Требуется HuggingFace токен для speaker diarization
